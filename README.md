# Hópverkefni 2
### Brynjar og Steinunn

### Keyrsla

Verkefnið er keyrt með:

```bash
npm install
```

 og svo: 

- `npm run dev` til að keyra í þróunarumhverfi með engu cache
- `npm run build` til að búa til keyrsluútgáfu
- `npm run lint` til að keyra stylelint og eslint

### Uppsetning verkefnis

Við í þessum hóp trúum á að það er skipulagning í óreiðu og erum því ekki með skrár skiptar niður í möppur.

Fyrir lítið verkefni eins og þetta er líka þæginlegt að vera ekki að flækja hluti. 

Verkefnið notar 3 .html skrár, eina .css skrá, eina .js skrá (og fleira).

### Útlistun á hvað var útfært

Verkefnið inniheldur eftirfarandi:

- README eftir forskrift, tæki og tól uppsett, vefur keyrir á Netilfy. Lint fyrir CSS/Sass og JavaScript.
- Git notað og PR eftir forskrift.
- Almennt útlit og skalanleiki útfært fyrir forsíðu og efnissíður.
- Slóðir og síður fyrir hvert efni.
- Grunvirkni útfærð.
- Valið efni útfært.

Fyrir valið efni var valið:
    - Flóknari birting spurninga
    - Búa til spurningar og vista í localStorage
